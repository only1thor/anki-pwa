## State Management

- **Local State**: `useState` for component-level state
- **Database State**: `useLiveQuery` for reactive IndexedDB queries
- **No Redux/Context**: Simple enough without global state management

## Routing

Simple view-based routing using state:

```typescript
type View = 
  | { type: 'list' }
  | { type: 'editor'; deckId: string }
  | { type: 'review'; deckId: string };
```

No react-router needed for this simple app.

## Extending the Project

### Adding a New Component

1. Create file in `src/components/`
2. Import types from `src/types/`
3. Use database functions from `src/lib/db/`
4. Add to routing in `src/App.tsx`

### Adding a New Database Table

1. Update schema in `src/lib/db/index.ts`
2. Increment version number
3. Add type to `src/types/index.ts`
4. Add CRUD operations in `src/lib/db/index.ts`

### Adding a New Feature

1. Update types if needed
2. Add database operations
3. Create/update components
4. Update documentation
5. Test thoroughly
